## 🔮 기획의도

넘쳐나는 글자 속에서 원하는 정보를 얻기 힘들어진 요즘<br/>
다른 사람의 TMI가 아닌, 원하는 정보를 직관적으로 파악하도록 돕기위하여 기획

<br/>


## ⚙️ 기능

- 직관적 정보 제공
  - 마인드맵을 이용한 게시글 작성 
- 소통을 통한 공동체의식 부여
  - 마인드맵을 다른사람과 공동 수정 가능
  - 팔로우, 팔로잉
  - 스크랩
  - 댓글 작성 
- 좋아요, 스크랩을 통한 MIND 저장 등
